# Superpowers 技能库详解

## 一、定位

Superpowers 是面向 Claude Code、Cursor、Codex 等 AI 编程代理的**完整软件开发工作流系统**。它通过一套可组合的「技能」（Skills）让 AI 自动遵循最佳实践，而非随意编程，使 AI 表现得像有经验的工程师而非实习生。

- **核心理念**：测试驱动、系统化流程、复杂度削减、证据验证

---

## 二、五阶段工作流

| 阶段 | 内容 |
|------|------|
| 1. 头脑风暴 | AI 通过苏格拉底式对话厘清需求 |
| 2. 工作区隔离 | 创建独立的 git 分支和 worktree |
| 3. 编写计划 | 将设计拆解成 2～5 分钟的小任务 |
| 4. 子代理驱动开发 | 派遣新鲜子代理执行任务，并进行两阶段审查 |
| 5. 收尾 | 验证测试、清理环境 |

---

## 三、技能库内容

项目包含 14+ 核心技能，涵盖：

- **测试驱动开发（TDD）**：先写测试再写实现，一次通过零报错
- **系统化调试**：四阶段根因定位流程
- **代码审查请求与响应**：标准化 Code Review 流程
- **并行代理工作流**：多子代理协同
- **计划编写与执行**：任务拆解与执行跟踪

---

## 五、典型使用流程

1. **需求澄清**：使用头脑风暴技能，与 AI 苏格拉底式对话明确需求
2. **工作区准备**：自动创建分支和 worktree，避免污染主线
3. **任务拆解**：AI 生成可执行的小任务清单
4. **子代理执行**：每个任务由「新鲜」子代理执行，减少上下文污染
5. **验证与收尾**：自动运行测试、生成报告、清理环境

---

## 六、与 Claude-Mem 的互补

- **Claude-Mem**：解决「跨会话记忆」问题，适合长期项目、调试、团队协作
- **Superpowers**：解决「单次任务流程」问题，强化 TDD、计划、子代理等工程实践

两者结合可显著提升 Claude Code 在复杂开发场景下的表现。
